Get Risoluto running and process your first issue in under 10 minutes.

## Prerequisites

| Requirement | Details |
|-------------|---------|
| **Node.js** | v22 or newer |
| **Docker** | Docker Engine running (`docker info` should succeed) |
| **pnpm** | Package manager (`npm install -g pnpm`) |
| **Linear** | A Linear account with at least one project |

## Install & Start

<Steps>
  <Step title="Clone and build" icon="download">
    <CodeGroup>
      ```bash pnpm (recommended)
      git clone https://github.com/OmerFarukOruc/risoluto.git
      cd risoluto
      pnpm install && pnpm run build
      ```

      ```bash npm
      git clone https://github.com/OmerFarukOruc/risoluto.git
      cd risoluto
      npm install && npm run build
      ```
    </CodeGroup>
  </Step>

  <Step title="Build the sandbox image" icon="docker">
    This creates the Docker image that isolated AI agents run inside:

    ```bash
    bash bin/build-sandbox.sh
    ```

    <Info>
      The sandbox image is based on **Ubuntu 24.04 with Node.js 22** and the Codex CLI pre-installed. It provides process isolation, resource limits, and security hardening out of the box.
    </Info>
  </Step>

  <Step title="Start Risoluto" icon="play">
    ```bash
    node dist/cli/index.js --port 4000
    ```

    Open [http://127.0.0.1:4000](http://127.0.0.1:4000) — the **setup wizard** opens automatically on first run.
  </Step>
</Steps>

## Setup Wizard

The wizard walks you through four steps:

<Steps>
  <Step title="Protect secrets" icon="key">
    Generates an AES-256 encryption master key that protects all stored credentials on disk. This key never leaves your machine.
  </Step>

  <Step title="Connect Linear" icon="plug">
    Paste your Linear API key. The wizard validates it, lists your projects, and lets you select the one to monitor.
  </Step>

  <Step title="Add OpenAI" icon="brain">
    Choose one of three authentication methods:

    <Tabs>
      <Tab title="API Key">
        Paste an `sk-...` API key directly. Risoluto validates and encrypts it.
      </Tab>
      <Tab title="Browser Sign-In">
        Click **"Sign in with OpenAI"** — authenticates via PKCE in your browser. No CLI needed.
      </Tab>
      <Tab title="Manual">
        Run `codex login` in your terminal, then paste the contents of `~/.codex/auth.json`.
      </Tab>
    </Tabs>
  </Step>

  <Step title="Add GitHub (optional)" icon="github">
    Paste a GitHub PAT to enable automatic PR creation when agents complete work.
  </Step>
</Steps>

## Verify It Works

Create a test issue in your Linear project:

**Title:**
```
SMOKE: create workspace proof file
```

**Description:**
```markdown
Create `RISOLUTO_SMOKE_RESULT.md` in the workspace with:
- the issue identifier
- the current UTC timestamp
- the output of `pwd` and `ls -la`
- whether the workspace looks empty or repo-backed
```

Set the issue to **In Progress**. With webhooks configured, Risoluto picks it up in real time. Otherwise, it detects the issue within one poll cycle (default: **15 seconds**).

<Check>
  **How to confirm it's working:** The issue appears on the dashboard at [http://127.0.0.1:4000](http://127.0.0.1:4000) with real-time progress indicators. You can also check the API directly.
</Check>
